# ThreadPrinter Icons

This directory contains the extension icons in the following sizes:

- `icon16.png` - Toolbar icon
- `icon48.png` - Extension management page
- `icon128.png` - Chrome Web Store

## Icon Design Guidelines

- Simple, recognizable design
- Should work on both light and dark backgrounds
- Use the brand color: #1D9BF0 (Twitter Blue)

## Temporary Icons

Until custom icons are created, you can use any placeholder images or generate simple icons using online tools like:

- [Favicon.io](https://favicon.io/)
- [Canva](https://www.canva.com/)
- [Figma](https://www.figma.com/)

Or use a simple SVG to PNG converter for the provided icon design.
